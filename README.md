# yyw 使用说明

yyw 是一个 Windows 本地音频处理工具。它可以扫描网易云下载目录中的 `.ncm` 和常见音频文件，转换 NCM，分离人声/伴奏/鼓/贝斯等音轨，并在软件里直接播放结果。

## 适合谁使用

如果你想做这些事，可以使用 yyw：

- 把网易云 `.ncm` 转成可处理的音频。
- 从歌曲里分离人声、伴奏或更多分轨。
- 试听原曲和分离后的音轨。
- 查看歌曲码率、时长、编码、采样率等信息。
- 查看封面和歌词。

## 快速启动

解压发布包后，推荐双击：

```text
start_yyw.bat
```

这个脚本会自动检查运行环境：

1. 检查电脑里是否有 Python。
2. 如果没有 Python，会尝试用 `winget` 安装 Python 3.11。
3. 检查是否安装 Demucs。
4. 如果没有 Demucs，会自动安装 `demucs` 和 `torchcodec`。
5. 启动 `yyw.exe`。

第一次运行可能需要较长时间，因为 Python、Demucs 和相关依赖需要联网下载。

如果你的电脑已经装好了 Python 和 Demucs，也可以直接双击：

```text
yyw.exe
```

## 发布包里有什么

正常发布包大致包含这些文件：

```text
yyw-release\
├── yyw.exe              # 主程序
├── yyw-cli.exe          # 命令行版本，普通用户可忽略
├── start_yyw.bat        # 推荐启动脚本
├── README.md            # 本说明文件
├── separators.json      # 分离工具配置
├── radian.png           # 图标资源
└── tools\
    ├── ncmdump.exe      # NCM 转换工具
    ├── ffmpeg.exe       # 媒体信息、封面、歌词、时长辅助工具
    └── *.dll            # FFmpeg 运行库
```

请不要随意删除 `tools` 目录，否则 NCM 转换、封面读取、媒体信息读取可能无法使用。

## 基本使用流程

1. 打开 `start_yyw.bat` 或 `yyw.exe`。
2. 在“音频目录”选择你的歌曲目录，例如网易云下载目录。
3. 在“输出目录”选择分离结果保存位置，默认是 `stems_output`。
4. 点击“扫描”。
5. 在左侧选择要处理的歌曲。
6. 选择分离模式、模型和设备。
7. 点击“转换 NCM”或“分离选中”。
8. 等待底部进度条完成。
9. 在右侧“输出音轨”卡片中双击音轨试听。

## 分离模式怎么选

常用模式：

- `vocals`：分离人声和伴奏，最常用。
- `four_stems`：分离人声、鼓、贝斯、其他。
- `six_stems`：分离人声、鼓、贝斯、吉他、钢琴、其他。

常用模型：

- `htdemucs`：默认模型，适合大多数情况。
- `htdemucs_ft`：更精细，但可能更慢。
- `htdemucs_6s`：建议配合 `six_stems` 使用。

设备选择：

- `auto`：自动选择，推荐。
- `cpu`：只用 CPU，速度较慢但兼容性更好。
- `cuda`：使用 NVIDIA 显卡加速，需要电脑支持 CUDA。

## 播放音轨

右侧“输出音轨”区域会以卡片显示分离结果。

- 单击卡片：选中音轨。
- 双击卡片：直接播放音轨。
- 选中卡片后，也可以点击底部播放器的“播放音轨”。
- 右键卡片：可以播放、查看详细信息、打开所在目录。

底部播放器支持：

- 播放输入音频。
- 播放输出音轨。
- 暂停、继续、停止。
- 查看当前时间和总时长。
- 拖动进度条跳转。
- 显示封面和歌词。

## 查看歌曲信息

在左侧歌曲列表或右侧音轨卡片上右键，选择“查看详细信息”。

可以看到：

- 文件大小
- 时长
- 总体码率
- 音频编码
- 采样率
- 声道数
- 封面流信息

这些信息依赖发布包中的 `tools\ffmpeg.exe`。

## 歌词和封面

yyw 会尝试自动读取：

- 音频内嵌封面
- 音频内嵌歌词
- 同名图片，例如 `歌曲名.jpg`、`歌曲名.png`
- 常见封面文件，例如 `cover.jpg`、`folder.jpg`
- 同名歌词，例如 `歌曲名.lrc`、`歌曲名.txt`

网易云扩展歌词中的 JSON 行会被自动清理成普通歌词文本。

## NCM 转换

如果扫描到 `.ncm` 文件，yyw 会尝试使用：

```text
tools\ncmdump.exe
```

转换成功后，列表中会显示“已找到 FLAC”或“已找到 MP3”。

如果 NCM 没有自动转换，请检查：

- `tools\ncmdump.exe` 是否存在。
- 源目录是否有写入权限。
- 同名音频是否已经存在。

## 常见问题

### 第一次启动很慢

第一次运行 `start_yyw.bat` 可能会安装 Python、Demucs 和 torchcodec。它们体积较大，需要联网下载，请耐心等待。

### 提示找不到 Demucs

可以先运行 `start_yyw.bat` 自动安装。

如果仍然失败，可以手动打开命令行执行：

```powershell
python -m pip install demucs torchcodec
```

安装完成后重新打开 yyw。

### 分离速度很慢

如果使用 CPU 分离，速度会比较慢。电脑有 NVIDIA 显卡并正确安装 CUDA 环境时，可以尝试把设备改成 `cuda`。

### 右侧没有输出音轨

先确认：

- 分离任务已经完成。
- 输出目录设置正确。
- 点击过“刷新输出”。

右侧音轨以卡片显示，双击卡片即可播放。

### 封面或歌词没有显示

不是所有音频都带封面或歌词。你也可以把同名 `.lrc`、`.txt`、`.jpg`、`.png` 放在歌曲旁边，yyw 会尝试读取。

### 媒体信息读取失败

确认发布包中存在：

```text
tools\ffmpeg.exe
```

以及 `tools` 目录中的 FFmpeg DLL 文件。

## 命令行版本

普通用户通常不需要使用命令行版本。

如果你需要批处理，可以运行：

```powershell
.\yyw-cli.exe --help
```

示例：

```powershell
.\yyw-cli.exe -i "G:\CloudMusic\VipSongsDownload" -o ".\stems_output" -M vocals
```

## 隐私说明

yyw 在本地运行。歌曲扫描、分离、播放、封面和歌词读取都在你的电脑上完成。

`start_yyw.bat` 只有在安装 Python/Demucs 依赖时需要联网。
